# Synergy Software/ML Taskphase

This repository contains the completed Software/ML taskphase work.

## Tasks

- `task_1`: Development environment and Linux basics
- `task_2`: Python fundamentals and submission analysis
- `task_3`: Manual CSV parsing and pandas comparison
- `task_4`: Data cleaning and validation
- `task_5`: Matplotlib data visualization
- `task_6`: Conceptual report covering Tasks 1–5

## Setup

Create and activate a Python virtual environment, then install the dependencies:

```bash
python -m venv .venv
```

Windows PowerShell:

```powershell
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Run each task from the repository root using the command in its README.
